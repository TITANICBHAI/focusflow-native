FocusFlow
by TBTechs

Your focus data stays on your PC.

FocusFlow for Windows is a desktop productivity app that enforces focus sessions by blocking distracting applications. This policy explains what the app stores, what system access it uses, and how your data is handled.

Last updated: May 2025  ·  Applies to: FocusFlow for Windows (Desktop)

This Privacy Policy may be updated at any time without prior notice. Continued use of the app after any change constitutes your acceptance of the updated policy.

Local-first storage

FocusFlow stores all data locally on your Windows PC using a SQLite database file located in your user profile folder (%USERPROFILE%\.focusflow\). This includes:

  • Tasks, schedules, and recurring task rules
  • Focus session history, durations, and notes
  • App block rules, keyword blockers, and block schedules
  • Daily allowance settings and usage logs
  • Habit tracking data and streaks
  • Daily notes and app settings

No data is transmitted to or stored on any FocusFlow-operated or TBTechs-operated server.

No account system or backend

FocusFlow does not require a user account, email address, or any form of registration. There is no FocusFlow backend service, cloud sync, or remote database. The app operates entirely offline and locally on your device.

Windows system access used

  • Process enumeration: FocusFlow reads the list of running Windows processes to detect and close blocked applications during focus sessions or always-on enforcement. It does not read process memory, file contents, or private data from other applications.
  • Win32 API (JNA): Used to send close signals to blocked application windows. No keystroke logging, screen recording, or input capture is performed.
  • System tray: Used to display a tray icon and provide quick controls (restore window, toggle blocking, quit) while the app runs in the background.
  • File system (user profile only): Used to read and write the local SQLite database and crash log in %USERPROFILE%\.focusflow\. FocusFlow does not access files outside this directory.
  • Auto-backup: FocusFlow creates periodic rolling backups of your local database within the same .focusflow folder. These backups never leave your device.

What FocusFlow does not collect

  • No location data
  • No camera or microphone access
  • No passwords, clipboard contents, or private message contents
  • No payment data
  • No usage analytics sent to any server operated by FocusFlow or TBTechs
  • No crash reports transmitted externally (crash logs are written locally only)

Windows App Package (MSIX)

The Microsoft Store version of FocusFlow is distributed as an MSIX package. The app declares the runFullTrust capability, which is required to enumerate running processes and enforce app blocking on Windows. This capability is not used to collect, transmit, or store any data beyond what is described in this policy.

Your control

You can uninstall FocusFlow at any time through Windows Settings → Apps. To remove all stored data, delete the %USERPROFILE%\.focusflow\ folder after uninstalling. No data remains on any server because none is ever transmitted there.

Children's privacy

FocusFlow does not knowingly collect any data from anyone. Since no personal data is collected or transmitted, the app is safe for use regardless of age. Parents may use FocusFlow to manage focus sessions on shared devices.
