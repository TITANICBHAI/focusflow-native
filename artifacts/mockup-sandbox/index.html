FocusFlow (by TBTechs) is a hard-enforcement screen time management and app blocking suite for Android, Windows, and macOS. It combines three stacked enforcement layers — Android Accessibility Service, a local null-routing VPN, and Android Device Administrator — with behavioral deterrents, a scheduling engine, and commitment tools like Nuclear Mode that no other productivity app provides. It is free.

Why Soft Blocking Fails

  • Americans spend an average of 4 hours 37 minutes per day on their smartphones (Statista Mobile Usage Report, 2024).
  • Knowledge workers lose approximately 2 hours of productive work per day to smartphone distractions (RescueTime Annual Productivity Report).
  • A 2021 study in the Journal of Experimental Psychology found 85% of smartphone users return to the same distracting app within minutes of closing it.
  • University of California, Irvine research found it takes an average of 23 minutes to regain full focus after a smartphone interruption.
  • Google Digital Wellbeing's soft timers can be dismissed with a single tap — making them ineffective for compulsive use patterns. FocusFlow cannot be bypassed this way.

Three Enforcement Layers

  1. Accessibility Service (AppBlockerAccessibilityService.kt — 3,394 lines) — monitors every app window, fires within milliseconds of detection, performs up to 5 rapid re-checks at 300ms intervals. Intercepts the Play Store install flow, uninstall dialogs, and Accessibility Settings navigation during active sessions.
  2. Local Null-Routing VPN (NetworkBlockerVpnService) — routes selected app traffic into a VPN that never forwards packets. PER_APP mode: only the blocked app loses internet. GLOBAL mode: all internet cut except emergency calls. No traffic leaves the device.
  3. Device Administrator (FocusDayDeviceAdminReceiver) — blocks uninstallation via Settings. System Guard intercepts Clear Data, Accessibility Settings, and Uninstall screens. SHA-256 native PIN gates all stop-session operations at the native layer.

Who Uses FocusFlow

FocusFlow serves users who have already tried soft-timer apps and found them insufficient. Common use cases:

  • Phone addiction recovery — Nuclear Mode permanently removes addictive apps; Temptation Log shows weekly blocked-attempt data
  • Students (deep study blocks) — Task-based focus whitelists only study tools; Pomodoro timer signals work/break phases automatically
  • Professionals (work-hours blocking) — Recurring Block Schedules block social media Mon–Fri 09:00–18:00 without any active session; VPN blocking can be added per-schedule
  • Digital minimalists — Always-On mode enforces a permanent blocklist; home screen launcher shows only whitelisted apps
  • ADHD self-management — Per-task allowed app lists, aversive feedback as external interrupt, Pomodoro work/break cycles, chronotype-aware scheduling
  • Self-control (parental-style) — Device Admin + SHA-256 PIN makes the block effectively irreversible during a session, even for the person who set it up

Pomodoro Timer and Task Manager

  • Integrated Pomodoro timer (usePomodoro.ts) — configurable work and break durations; fires vibration + push notification at each phase transition ("Break Time" / "Back to Work")
  • App blocking active throughout — blocked apps remain blocked during breaks; Pomodoro does not unlock the block list
  • Per-task allowed app list — each task in the built-in task manager defines its own whitelist (focusAllowedPackages): a study task allows only a dictionary; a work task allows a messaging app
  • Task priorities — low, medium, high, critical. Three reminder types: pre-start, at-start, post-start
  • Chronotype-aware scheduling — user profile records peak focus time (morning/afternoon/evening/night) and break style preference (short frequent / balanced / long infrequent / no-break)

Import Block List from Other Apps

Switching to FocusFlow from AppBlock, StayFree, Digital Wellbeing, Stay Focused, or Lock Me Out? FocusFlow's import feature (871-line ImportFromOtherAppModal) handles it in one step:

  • File import — auto-detects TXT (one package name per line), CSV (any column with valid package names), and JSON (arrays or objects with packageName/package/appId keys)
  • Name-based import — type or paste app display names ("Instagram", "TikTok"); FocusFlow fuzzy-matches against installed apps and shows a preview before adding to the block list
  • Works with exports from: AppBlock, StayFree, ActionDash, Digital Wellbeing Takeout. Works without export for: Stay Focused, Lock Me Out (free tier)

Weekly Analytics and Clean Streak

  • Day-by-day blocked attempt chart — bar chart of blocked-app attempts for each of the last 7 days
  • Top tempted apps — apps sorted by blocked-attempt count for the current week
  • Clean Streak — consecutive days with zero blocked-app attempts. A clean day means you never once opened a blocked app, even instinctively
  • All-time total — lifetime blocked-attempt count vs. this week's total

Privacy and Technical Notes

  • No root required — works entirely within standard Android permissions (Accessibility Service, VPN, Device Admin)
  • No data leaves the device — local null-routing VPN never forwards packets; all data stored on-device in SharedPreferences and SQLite
  • Dark mode — defaults to dark; user can toggle independently of system setting
  • Backup and restore — exports a .focusflow file using ACTION_CREATE_DOCUMENT; restores all tasks, settings, block lists, and schedules
  • FLAG_SECURE — sensitive screens (PIN entry, session controls) prevent screenshots and screen recordings

Frequently Asked Questions

What is FocusFlow app?

FocusFlow (by TBTechs) is a hard-enforcement screen time management app for Android, Windows, and macOS. It blocks distracting apps and websites using Android Accessibility Services, a local VPN, and Device Administrator mode. It is unrelated to other apps sharing the FocusFlow name — TBTechs' version is a hard-enforcement blocking suite, not a simple Pomodoro timer.

Does FocusFlow require root?

No. FocusFlow works entirely within standard Android permissions — Accessibility Service, VPN, and Device Administrator. No rooting or system modification is required on any device.

Does FocusFlow have a Pomodoro timer?

Yes. FocusFlow includes a built-in Pomodoro timer integrated with app blocking. Work and break durations are configurable. The timer fires push notifications and vibration at each phase transition. App blocking remains active during breaks — breaks do not unlock blocked apps.

Can I import my block list from AppBlock, StayFree, or Digital Wellbeing?

Yes. FocusFlow auto-detects TXT, CSV, and JSON exports from AppBlock, StayFree, ActionDash, and Digital Wellbeing. For apps without export (Stay Focused, Lock Me Out), paste app names and FocusFlow fuzzy-matches them against installed apps.

Is FocusFlow good for phone addiction?

The Temptation Log records every blocked-app attempt and shows weekly data. Nuclear Mode permanently removes addictive apps. Aversive feedback (screen dim to 2% + vibration) creates a negative stimulus associated with reaching for a blocked app. The Weekly Report surfaces behavioral patterns over time.

How does FocusFlow prevent you from uninstalling it?

FocusFlow requests Android Device Administrator privileges via FocusDayDeviceAdminReceiver. Once granted, Android blocks the standard uninstall path. The System Guard feature also intercepts navigation to the Uninstall, Clear Data, and Accessibility Settings screens during active sessions. A SHA-256 native PIN gates all stop-session operations.

Can FocusFlow block YouTube Shorts without blocking all of YouTube?

Yes. FocusFlow's YouTube Shorts Blocker closes the Shorts player while leaving the main YouTube feed, search, subscriptions, and video playback accessible. Instagram Reels blocking works the same way.

Does FocusFlow work after a phone reboot?

Yes. FocusFlow includes a BootReceiver that automatically restarts the blocking service and VPN tunnel if the device reboots during an active session, closing the most common bypass method.

How is FocusFlow different from Freedom and Digital Wellbeing?

Freedom uses Accessibility Service and VPN but lacks Device Admin, boot recovery, and aversive feedback. It costs $39.99/year. Digital Wellbeing uses soft timers dismissable with one tap, has no VPN enforcement, and cannot prevent its own uninstallation. FocusFlow combines all three enforcement layers and is free.
